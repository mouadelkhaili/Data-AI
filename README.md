# 🎓 Workshops Centrale Tech - Data & AI

Bienvenue sur le repository officiel des séances de Workshop du pôle Data & AI de **Centrale Tech** ! 

Ce repository contient tous les notebooks, datasets et ressources pédagogiques pour les ateliers pratiques de découverte de la data science et de l'intelligence artificielle.

## 📚 Contenu des Workshops

Vous trouverez dans ce repository :
- 📊 Notebooks Jupyter interactifs pour l'exploration de données
- 🤖 Tutoriels de Machine Learning et Deep Learning
- 📈 Exercices de visualisation de données avec Matplotlib, Seaborn
- 🐼 Manipulation de données avec Pandas
- 🧮 Régression linéaire et modèles prédictifs
- 📁 Datasets pour la pratique

## 🚀 Guide de Démarrage Rapide

### 1️⃣ Créer un Compte GitHub

Si vous n'avez pas encore de compte GitHub, suivez ces étapes :

1. **Rendez-vous sur** [github.com](https://github.com)
2. **Cliquez sur** "Sign up" en haut à droite
3. **Remplissez le formulaire** :
   - Entrez votre adresse email
   - Créez un mot de passe sécurisé
   - Choisissez un nom d'utilisateur
4. **Vérifiez votre email** en cliquant sur le lien reçu
5. **Configurez votre profil** (optionnel mais recommandé)

✅ Votre compte GitHub est prêt !

---

### 2️⃣ Forker ce Repository

Le "fork" vous permet de créer votre propre copie du repository sur votre compte GitHub.

#### Méthode Web (Recommandée pour les débutants)

1. **Connectez-vous** à votre compte GitHub
2. **Visitez** le repository des workshops : `https://github.com/CentraleTechClub/Data-AI.git`
3. **Cliquez sur le bouton "Fork"** en haut à droite de la page
   
   <img src="https://docs.github.com/assets/cb-40742/mw-1440/images/help/repository/fork-button.webp" width="400">

4. **Sélectionnez votre compte** comme destination du fork
5. **Attendez** quelques secondes - votre copie est créée !

✅ Vous avez maintenant votre propre version du repository sur votre compte !

---

### 3️⃣ Cloner le Repository sur Votre Ordinateur

#### Option A : Avec GitHub Desktop (Plus simple)

1. **Téléchargez et installez** [GitHub Desktop](https://desktop.github.com/)
2. **Connectez-vous** avec votre compte GitHub
3. **Cliquez sur** "File" → "Clone Repository"
4. **Sélectionnez** votre fork dans la liste
5. **Choisissez** le dossier de destination sur votre ordinateur
6. **Cliquez sur** "Clone"

#### Option B : Avec Git en ligne de commande

```bash
# Cloner votre fork
git clone https://github.com/VOTRE-USERNAME/Data-IA.git

# Naviguer dans le dossier
cd Data-IA
```

---

### 4️⃣ Travailler sur les Notebooks

1. **Installez Jupyter** si ce n'est pas déjà fait :
   ```bash
   pip install jupyter pandas matplotlib seaborn numpy scikit-learn
   ```

2. **Lancez Jupyter** :
   ```bash
   jupyter notebook
   ```

3. **Ouvrez les notebooks** (fichiers `.ipynb`) et commencez à coder ! 🎉

---

### 5️⃣ Sauvegarder Vos Modifications

#### Avec GitHub Desktop

1. **Ouvrez GitHub Desktop**
2. **Vérifiez les changements** dans l'onglet "Changes"
3. **Ajoutez un message** décrivant vos modifications
4. **Cliquez sur** "Commit to main"
5. **Cliquez sur** "Push origin" pour envoyer sur GitHub

#### Avec Git en ligne de commande

```bash
# Vérifier l'état de vos fichiers
git status

# Ajouter tous les fichiers modifiés
git add .

# Créer un commit avec un message descriptif
git commit -m "Complétion du workshop 1 - Data Exploration"

# Envoyer les modifications sur GitHub
git push origin main
```

✅ Vos modifications sont maintenant sauvegardées sur votre GitHub personnel !
